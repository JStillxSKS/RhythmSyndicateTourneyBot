# RS Visual Editor (text only)

Change words on the official mockup cards and save a PNG.

## On a computer (supported)

1. Open folder: `RhythmSyndicateTourneyBot\visual-editor\`
2. Double-click **`index.html`** (Edge / Chrome)
3. Edit text → **Download PNG**

**Phone:** not a goal for now — leave it on PC.

First load needs internet once (`html2canvas` CDN). Keep **`logo.jpg`** next to `index.html`.

Save PNG → upload to channel (or hand to staff for `/rs` posts).
